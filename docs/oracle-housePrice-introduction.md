# 介绍

## 概述

预言机合约提供美国十大城市近一年房产价格数据。

数据即可按月份为单位，一次性返回近一年的房价数据。又可以根据月份参数单独返回距今为止某月份数据。

- 获取全部房价数据
- 按月获取房价数据

## 支持城市

| 序号 | 城市          |
| ---- | ------------- |
| 1    | New York      |
| 2    | Los Angeles   |
| 3    | Chicago       |
| 4    | Houston       |
| 5    | Philadelphia  |
| 6    | Detroit       |
| 7    | San Francisco |
| 8    | Boston        |
| 9    | Pittsburgh    |
| 10   | Atlanta       |

## 技术支持

- 百万级数据库 ：持续更新全美城市各区房价信息
- 安全稳定的喂价机制：[分布式数据模式]() 架构 
- 同算法 [HTTP API]() 数据查询服务：保证价格信息与预言机内容无偏差
